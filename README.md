 Evaluation-Gestion-ParcAutomobile
 Description
Projet Java permettant de gérer des véhicules (voitures, camions), des clients et des locations.

Contenu du projet
`Vehicule`, `Voiture`, `Camion` → Héritage
`Client` → Infos utilisateur
 `Date` → Gestion des dates
`Location` → Associe Client + Véhicule + Dates
 `ParcAuto` → Liste des véhicules
`Main` → Tests et affichages
 Fonctionnalités
 Ajouter des véhicules dans le parc
 Lister les véhicules disponibles
 Louer et rendre un véhicule
Calculer le prix d'une location
Afficher les détails d'une location
